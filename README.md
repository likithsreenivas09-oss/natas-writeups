# OverTheWire Natas Writeups

This repository contains my notes and writeups while solving the OverTheWire Natas web security wargame.

## About Natas

Natas is a web security wargame that teaches:

- Viewing page source
- Hidden files and directories
- Client-side security weaknesses
- Cookies and sessions
- HTTP headers
- Command injection
- File inclusion
- SQL injection
- Authentication bypass
- Other web security concepts
