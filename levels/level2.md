# Natas Level 2

## Goal
Get the password for Natas Level 3.

## Procedure
1. Accessed the website using the credentials given.
2. Inspected the page source for any clues.
3. Noticed that the website contained files that were not accessible from the website’s page.
4. Went through the hidden location and accessed the password.

## Knowledge Acquired
Web pages can contain files that are not linked to in the web page itself.

## Lesson in Cybersecurity
Files that are sensitive should not be easily accessible, regardless of whether or not they are linked to.
