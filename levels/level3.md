## Concept Learned

Websites can use a standard file called `robots.txt` to give instructions to search engines about which pages or directories should not be indexed or crawled. This is commonly used to avoid indexing unnecessary pages and to reduce unnecessary crawler traffic.

## Security Lesson

`robots.txt` is not a security mechanism. The file is publicly accessible, and anyone can read it to discover hidden directories or files. Sensitive information should be protected using proper authentication and permissions, not by hiding it from search engines.
