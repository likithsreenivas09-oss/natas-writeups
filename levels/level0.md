# Natas Level 0

## Objective
Obtain the password for Natas Level 1.

## Process
1. Accessed the website using the login details given by OverTheWire.
2. Used `Ctrl + U` to see the source code of the webpage.
3. Located the password in the HTML comment.

## Concept Gained
Data that is being sent to the browser is still viewable even though it may not be visible in the webpage.

## Security Consideration
Do not keep any important data such as passwords in HTML comments since the page source can be seen by users.
