# Natas Level 1

## Objective
Get the password for Natas Level 2.

## Process
1. I logged into the website with the credentials provided by OverTheWire.
2. I used "Ctrl + U" to view the source code of the web page.
3. I located the password in the source code of the HTML file.

## Concept Learned
Client-side restrictions like right-clicking can be easily bypassed.

## Security Takeaway
It is not secure to hide any information using JavaScript or other browser controls.
